3. Usage

In this section, we will explain you the basis to actually be able to play Boson. The first thing to know is that, because of the lack of any AI player (artificial intelligence), you won't be able to play alone. You need to have at least another player to play against.

3.1 Launching Boson

When you first launch boson (usually by clicking on the K-menu/Game/Boson), you are presented with a blank window.

Every player uses the connect to Server item in the file menu.

One (and only one) of the players have to start a server, the name of the server will then be the name of its computer.

Other players have to enter the name of the server in the dialog box, and then hit connect

3.2 User interaction

On the keyboard, you may use the arrow keys to move across the map.

The left button of the mouse let you select objects Left-click on a facility or a mobile unit to select it. If you left-click on the ground and drag the mouse (while still having the button down), it will draw a selection rectangle that will select everything in it when you release the button.

When a unit or a facility is selected, the lower bar is as big as is the energy of the selected item. The more red it is, the less energy it has. When the energy becomes null, the item is destroyed.

The right-button let you do actions : on the ground it will make a mobile unit to move there, on a friend mobile unit, it will follow it, on a ennemy, it will attack it.

The middle-button is used to move across the map. By middle-clicking on any place, the point you've clicked on will become the new center of the map. If you click on the center, it will do nothing. If you click on the border, you'll make a big jump.

3.3 The editor

When using righ-click to put some ground on the map, you may let the mouse clicked while moving the mouse, it will fill the map with the current ground tile. If you have the shift key pressed while doing so, you will be in a special big-fill mode, that is quiker to fill big places.
