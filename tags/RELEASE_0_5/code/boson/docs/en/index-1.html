1. Introduction

A RTS (real-time strategy game) was missing under linux, and we felt like we could do it. KDE/Qt appeared to be the best environment under which such a game could be done.

We hope you will enjoy this game as much as we did while doing it and we would appreciate any suggestions and comments.

Please contact : Thomas Capricelli orzel@yalbi.com.

1.1 Changes

Boson 0.5 is the fifth alpha release, it's meant to be one of the last developper-only release. It's still no yet intended for playing, though.

New features in this release are :

  • Port from Qt-1.x to Qt-2.1
  • Port from KDE-1.x to KDE-2.0pre (at least 1.90)
  • Port from QwSpriteField to QCanvas (for rendering)
  • You may open several windows in boson or the editor. Seeing the same map.
