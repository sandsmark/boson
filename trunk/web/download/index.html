Download Boson:

Please note that Boson isn't playable yet, so releases are made available mainly for developers, artists and anybody else just being curious. Nevertheless, if you encounter problems during the installation or in the game, please report this in the feedback section. Thank You.

This section will become more complete as things like additional themes (graphics, music and sounds) are added.



RPMs and SRPMs:

Boson is in a development state and I am far too lazy to generate RPMs right now. Please download the sources instead. If you *really* want/need RPM you might want to contact us at boson-devel@lists.sourceforge.net.


Sources:

We currently do not provide packages for the source code. It would be too much work for us to debug obsolete packaged code, so please use the CVS code. You can get the code using these commands:

cvs -d:pserver:anonymous@cvs.boson.sourceforge.net:/cvsroot/boson login
cvs -z3 -d:pserver:anonymous@cvs.boson.sourceforge.net:/cvsroot/boson co code

please just press the Enter key when you are prompted for a password.
As soon as you have checked out the sources you can simply do a

cd code
cvs -z3 up -Pd

to update the sources. No need to login again.
To compile boson use something like

cd code
./configure --prefix=/opt/kde3
make

You need to be root to install boson:

su
make install



Graphics:

The graphics packages for Boson provide all the neccessary graphics files to play boson, including ground, species and GUI graphics. Just like for the sources we use CVS for these so-called data files. We don't provide packages at this time as there are quite often (very small) changes to some files and you would have to download the entire package again.
So please use CVS for this as well. When there are changed to the packages CVS only updates the changed parts.

Just use

cvs -d:pserver:anonymous@cvs.boson.sourceforge.net:/cvsroot/boson login
cvs -z3 -d:pserver:anonymous@cvs.boson.sourceforge.net:/cvsroot/boson co data

to checkout the data files (veery big package! currently about 24MB). Use

cd data
cvs -z3 up -Pd

to update the data files. To install them use something like

./configure --prefix=/opt/kde3
make
su
make install

See Sources for details - and don't forget to read the README file!
