General description :

  • A server controls the game, clients connect to it. Server and client(s) can run on the same computer.
  • It uses the KDE libraries and will be KDE aware (menu, config).
  • The graphics for the grounds, units, facilities and others are themeable.
  • The sounds will be themeable.
  • Synchronous communication, while not real-time (ever tried to do RT over TCP/IP ?).
  • Big terrain, presently already up to 200x200 sprites.

 
Commands :

Although these things might still change, it might be helpful to know how to issue some of the commands already existing in Boson. Right now, you can / must use only the mouse. Later, we will add some keyboard shortcuts and maybe some kind of a toolbar.

Game
  • To select a unit or building, leftclick on it. You can select multiple units by holding down the left mousebutton and pulling a rectangle around them.
  • To construct a new building, select the command bunker, leftclick on the facility you want to build and leftclick in the view to place it.
  • To construct a new unit, select the warfactory, leftclick on the unit you want to build and leftclick in the view to place it.
  • To move a unit, select it and set its target by right-clicking on the map.
  • To attack your enemy, select your unit(s) and rightclick on the enemy's unit / building.
  • You can recenter the map by middleclicking on any point in the view.
Editor
  • When a simple ground (no transition) is selected, you may move the mouse while clicking to fill the field.
  • While filling you may hold the shift key : it will help you a lot I guess :-)
