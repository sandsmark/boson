3. Usage

In this section, we will explain you the basis to actually be able to play Boson. The first thing to know is that, because of the lack of any IA player (artificial intelligence), you won't be able to play alone. You need to have at least another player to play against.

3.1 Launching the server

The server is the central piece of the game, the place where everything is coordinated, and the link between the different players. It must be launched on one and only one computer. Let's say for the example that this computer is called 'eagle'

If you have successfully installed boson, you should have a program called boserver. This is the first program to launch so that every single player may connect to it.

You should run it in background, typing 

[eagle]% boserver&

You can also click on boserver, in the K-Menu, under the group Games.

Don't worry about the still naked interface of it, just hide it if you don't like it, there's nothing in the window, but it _does_ do a lot of things behing.

3.2 Launching Boson

Every single player should perform this step on his computer. There is no problem if one of them is on the same computer as the server. 

[gauss]% boson eagle&

By giving the eagle argument to boson, you tell him on which computer he can find the server in order to connect to it.

You can also click on boson, in the K-Menu, under the group Games, but doing this way you won't be able to give any argument, so that boson will try to connect to the same computer. You can do this only if the boserver has been launched on _your_ computer.

There will soon be icons-to-click-on-to-get-things-simplier, but until then, you need to performs all of those steps.

3.3 User input

On the keyboard, you may use the arrow keys to move across the map.

The left button of the mouse let you select objects Left-click on a facility or a mobile unit to select it. If you left-click on the ground and drag the mouse (while still having the button down), it will draw a selection rectangle that will select everything in it when you release the button.

The right-button let you do actions : on the ground it will make a mobile unit to move there, on a friend mobile unit, it will follow it, on a ennemy, it will attack it. Note that so far, only move is implemented :-)

The middle-button is used to move across the map. By middle-clicking on any place, the point you've clicked on will become the new center of the map. If you click on the center, if will do nothing. If you click on the border, you'll make a big jump.
