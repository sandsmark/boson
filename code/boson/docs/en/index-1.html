1. Introduction

A RTS (real-time strategy game) was missing under linux, and we felt like we could do it. KDE/Qt appeared to be the best environment under which such a game could be done.

We would like to thanx Warwick Allison ( warwick@troll.no) for his wonderful QwSpriteField library, which is used by Boson. We hope you will enjoy this game as much as we did while doing it and we would appreciate any suggestions and comments.

Please contact : Thomas Capricelli capricel@enst.fr.

1.1 Changes

Boson 0.1 is the first release, it's meant to be a developper-only release and isn't yet intended for playing.

This release features :

  • Network layer and synchronisation.
  • Map file structure.
  • Selection and mutliple selection (control+click)
  • Units/Mobiles displaying.
  • One can move units
  • Units can move only on 'allowed' ground and don't overlap.
