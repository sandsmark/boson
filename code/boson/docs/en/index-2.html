2. Installation

2.1 How to obtain Boson

Boson can be found on the boson homepage.

Until the first public release (should be 0.5), this is the only place you can find it. Then, it will probably also be available on the main kde ftp site, and perhaps on some other places.

2.2 Requirements

In order to successfully compile boson, you need kdelibs-1.0 and kdesupport-1.0, or the latest versions of libkdecore, libkdeui, libkfile, libkfm and libQwSpriteField. All required libraries can be found on ftp://ftp.kde.org/pub/kde/.

2.3 Compilation and installation

In order to compile and install Boson on your system, type the following in the base directory of the Boson distribution: 

% ./configure
% make
% make install

You will probably need to be root for the install stage.

Since Boson uses autoconf you should have not trouble compiling it. Should you run into problems please report them to the the author at Thomas Capricelli
