4. Questions and Answers

Nothing happens when I click on the K-menu/Games/boson item.

If you haven't installed the boson-pics-0.1 package, boson will die as soon as launched. Install the package. If it's still won't be launched, do it in a terminal emulator (kvt, konsole, xterm) and send me

  • The output of the program before dying
  • The file boson-client.log that should be in the directory where you typed boson

Please contact : Thomas Capricelli if you have any questions, we will be happy to add it here if it's a frequently asked one.
