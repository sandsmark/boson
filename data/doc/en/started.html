Getting started

System requirements:

  • XFree 4 server with OpenGL support
  • 3D accelerator card
  • Optional: Sound card

Before you are able to install Boson, make sure that the following things are installed and working correctly on your PC:

  • QT (at least version 3)
  • KDE (at least version 3.0)
  • kdegames/libkdegames - usually shipped with KDE
  • kdemultimedia - also shipped with KDE
  • libvorbis (for sound and music)
  • lib3ds (Units and building loading)
  • OpenGL - The OpenGL library (should be included in you Xfree Server)
Please note that since we do not yet provide precompiled binaries you will need to install the development packages of these libraries as well (the development packages usually have a -devel or -dev in their name).

Now you can download boson - for instructions see http://boson.eu.org/download.html.
You need at least the packages "code" and "data". "music" is optional.
