CVS - Bleeding edge development version

You can get the code using these commands:
cvs -d:pserver:anonymous@cvs.boson.sourceforge.net:/cvsroot/boson login
cvs -z3 -d:pserver:anonymous@cvs.boson.sourceforge.net:/cvsroot/boson co code
Please just press the Enter key when you are prompted for a password. As soon as you have checked out the sources you can simply do a
cd code
cvs -z3 up -Pd
to update the sources. No need to login again.

Compilation

To compile boson use something like
cd code
make -f Makefile.cvs
./configure --prefix=/opt/kde3
make
You need to be root to install boson:
su
make install

Graphics

The graphics packages for Boson provide all the neccessary graphics files to play boson, including ground, species and GUI graphics. Just like for the sources we use CVS for these so-called data files. We don't provide packages at this time as there are quite often (very small) changes to some files and you would have to download the entire package again. So please use CVS for this as well. When there are changes to the packages, CVS will only updates the changed parts.

To checkout the data files (veery big package! currently about 24MB) (you need to be already logged in, see the cvs section) :

cvs -z3 -d:pserver:anonymous@cvs.boson.sourceforge.net:/cvsroot/boson co data

First you need to link the admin dir from the code directory:

cd data && ln -s ../code/admin admin

Now install the graphics following exactly the same rules as for the code :

make -f Makefile.cvs
./configure --prefix=/opt/kde3
make
su
make install
See Sources for details - and don't forget to read the README file!
