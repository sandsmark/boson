Sources

You need two and only two of the available packages : one for the code, and one for the graphics. If you have bunzip2, download both packages in bunzip2 format. Else download both packages in gunzip format.

Boson-pics-0.1.tar should take place into $KDEDIR/share/apps/. As explained in the boson/INSTALL file, it will create a directory called boson.
