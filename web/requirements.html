Boson requires the following Hardware / Software:

  • Any Unix on which KDE/Qt is running.
  • KDE 3.x libraries or greater, see KDE site (current CVS version)
  • libkdegames from the kdegames package KDE site (current CVS version)
  • Qt 3.x, see Troll Tech site
  • A minimum resolution of 800 x 600 pixels.
  • A 16-bit deep screen (High Color) is probably better.
  • Network connection (local or internet). This is not necessary as computer players are possible, too (but not yet written ;-))
